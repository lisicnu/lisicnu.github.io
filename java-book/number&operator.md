## 基础数据类型

| 类型    |    描述    |  长度  |        取值范围        |
| :------ | :--------: | :----: | :--------------------: |
| boolean |   布尔型   |        |     true 或 false      |
| byte    |   字节型   | 1 byte |       -128 ~ 127       |
| char    |   字符型   | 2 byte |                        |
| short   |   短整型   | 2 byte | -$2^{16}$~ $2^{15}$ -1 |
| int     |    整型    | 4 byte | -$2^{32}$~ $2^{31}$ -1 |
| float   | 单精度浮点 | 4 byte |                        |
| long    |   长整型   | 8 byte |                        |
| double  | 双精度浮点 | 8 byte |                        |

**`null`** 特殊类型
